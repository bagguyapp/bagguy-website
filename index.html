Product systems behind the app

The current Bag Guy build includes backend-ready systems for product lookup, coupon matching, receipt rewards, provider onboarding, and Telegram work-bag automation.

API

Coupon connector architecture

Built for Ibotta/IPN, Inmar, Eagle Eye, Kroger Products API, UPC lookup providers, retailer feeds, and admin-uploaded deals.

OCR

Receipt-safe reward flow

The mobile app uses a stable `/api/scan/receipt` contract with receipt IDs, idempotent rewards, and clear backend-down messaging.

BOT

Telegram agent stack

GPT Bot Core packages work into approved bags, prepares Codex task files, and keeps shell execution disabled by default.
